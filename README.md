## 🚀 Getting started

Clone the project and access the folder

```bash
$ git clone https://github.com/tgcarvalho/podcastr.git && cd podcastr
```

Follow the steps below
```bash
# Install the dependencies
$ yarn

# Run the JSON server
$ yarn server

# Start the project
$ yarn dev
```
